# PEC3
Tic-tac-toe game state analyzer
C program that, given the representation of a tic-tac-toe board(txt file), indicates the state of the game: whether player ‘X’ has won, player ‘O’ has won, there is a draw, or the game is still in progress.

Example:
If the content of board1.txt file is:
X O O
- X -
- - X
The program has to display the following output per screen:
INPUT
FILENAME?
board1.txt
OUTPUT
X WINS
